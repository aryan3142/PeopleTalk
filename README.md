# PeopleTalk 
### A Web Application where you can: 
#### 1. Register
#### 2. Login
#### 3. Create/Edit your profile
#### 4. Search your peeps
#### 5. Chat

### Backend -> Java 
### Frontend -> Javascript,HTML,CSS
### Database -> MySQL, ORACLE
